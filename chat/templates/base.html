Welcome to my little chat app.

This project is a simple chat application built with Python. It allows users to create and participate in real-time chat rooms. To start a new conversation, a user simply navigates to `/chat/chatName`, where `chatName` represents the unique name of the new chat room, then share the link with a friend. Alternatively, you can click on the button below to generate a random group name for your chat room.

Get Started
Beginner Friendly

This chat app is designed with simplicity in mind allowing beginners to quickly understand and modify the app to suit their needs.

No Database

This project does not store data on a server or database. When the browser is closed, the chat history will be lost. This keeps the application lightweight and focused on real-time communication.
